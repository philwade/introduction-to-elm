Elm is an exciting new language for building fast and resilient front end web applications. For this talk,
I'll be giving a high level overview of Elm and an introduction to functional programming concepts.
Functional programming is that hot paradigm that that one person at your company won't shut up about
(hint: I'm that person.)  I'll talk about the practical reasons you might choose Elm and a functional
style and why it's fun! We'll go through basic language syntax and review the structure of an application or two.
